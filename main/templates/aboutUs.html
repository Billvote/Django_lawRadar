🚀 프로젝트 소개: LawRadar

📊 프로젝트 개요

LawRadar는 대한민국 20,21,22대 국회의원들의 주요 법안 표결 데이터를 성별·정당·클러스터별로 분석하고, 그 결과를 시각화하여 제공합니다. 선거구 위치 정보와 국회의원을 매핑하여 각 지역구 국회의원의 표결 현황을 쉽게 확인할 수 있도록 시각화했습니다. 데이터를 통해 사회적 갈등의 흐름과 정치적 경향을 더 쉽게 이해할 수 있도록 돕는 것이 목표입니다.

📚 데이터 출처

  • 열린 국회 API
  • 국회 의안정보시스템 (likms.assembly.go.kr)

제20대부터 제22대까지 국회 본회의에서 가결된 의안에 대한 표결 데이터를 수집했습니다. 수집된 데이터는 Python을 활용해 전처리한 후, 성별과 정당 정보를 매핑하여 분석에 활용했습니다.

📝 클러스터 분석 방식

의안별 '제안이유 및 주요내용' 텍스트를 수집하여, Kiwi 한글 형태소 분석기를 활용해 주요 명사를 추출했습니다. N-gram 벡터화한 내용을 기반으로 KMeans 알고리즘을 사용해 클러스터링을 진행했습니다. 최적의 클러스터 수는 실루엣 계수(Silhouette Score)를 기준으로 결정했으며, 각 클러스터는 핵심 키워드로 명명했습니다.

🛠️ 사용된 기술

  • 데이터 분석: Python (pandas, scikit-learn)
  • 백엔드: Django
  • 시각화: ApexCharts, Chart.js
  • 프론트엔드 스타일: Tailwind CSS
